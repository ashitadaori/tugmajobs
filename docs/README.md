# Documentation Structure

This directory contains all project documentation organized by category.

## Directory Structure

### `/features`
Feature implementation documentation and completion reports.
- New features added to the platform
- Feature completion summaries
- Feature-specific guides

### `/fixes`
Bug fixes and issue resolutions.
- Bug fix documentation
- Issue resolution reports
- Fix verification documents

### `/sessions`
Session summaries and work logs.
- Daily/weekly session summaries
- Work completion reports
- Progress tracking documents

### `/guides`
Comprehensive guides and verification reports.
- System documentation
- Implementation guides
- Verification and testing guides
- Architecture documentation
- Best practices

### `/debug`
Debug instructions and troubleshooting guides.
- Debug procedures
- Troubleshooting steps
- Issue investigation guides

## Quick Reference

### Finding Documentation
- **Feature docs**: Check `/features` for implementation details
- **Bug fixes**: Check `/fixes` for resolution steps
- **Session history**: Check `/sessions` for work summaries
- **How-to guides**: Check `/guides` for comprehensive documentation
- **Troubleshooting**: Check `/debug` for debug instructions
